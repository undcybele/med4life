Intro

Med4Life is an web app dedicated to improving the communication between doctors and patients. It helps a doctor easily see and analyse all of the important data about each of their patients.

Frontend

This app uses Angular2 for frontend and MongoDB for the database. The whole app is serverless and uses MongoDB atlas functions to easily store and retrieve data.
